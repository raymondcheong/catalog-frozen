# 部署 dashboard 與 Functions 到 Netlify

**問題**：產品目錄正常，但 dashboard.html 顯示 404 → 目前部署的版本沒有包含 dashboard.html。

**解決**：重新部署，讓 Netlify 取得最新的 `catalog-frozen` 內容（含 dashboard.html）。

> 📖 **新手請看**：詳細圖文步驟請參考 [新手部署教學.md](新手部署教學.md)

---

## 方式一：Netlify CLI（推薦，含 Functions）

### 1. 安裝依賴
```
cd c:\Users\admin\Desktop\CURSOR\catalog-frozen
npm install
```

### 2. 執行部署
雙擊 `deploy.bat`，或在終端機執行：
```
npx netlify-cli deploy --prod
```

### 3. 依提示操作（首次）
- 選 **「Link this directory to an existing site」**（用方向鍵選取後按 Enter）
- 在列表中選 **famous-manatee-f83275**
- 等待部署完成

### 若出現 EBUSY 錯誤
`deploy.bat` 已改為在 `C:\temp` 建立臨時目錄執行部署，避開 `C:\Users\admin` 的系統檔。請直接雙擊 `deploy.bat` 再試。

---

## 方式二：手動拖曳 ZIP（較簡單，但無 Functions）

若 CLI 無法使用，可用 ZIP 手動部署（僅靜態檔案，API 可能無法運作）：

1. 將 `catalog-frozen` **資料夾內的檔案**全部選取（index.html、dashboard.html、style.css、products.js 等）
2. 右鍵 → **壓縮成 ZIP**
3. 開啟 https://app.netlify.com → 進入站點 **famous-manatee-f83275**
4. 切到 **Deploys** 分頁
5. 將 ZIP 拖曳到部署區域

> 注意：ZIP 根目錄必須直接是 index.html、dashboard.html 等檔案，不要有 `catalog-frozen` 子資料夾。

---
